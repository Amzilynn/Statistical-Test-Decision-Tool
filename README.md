# Statistical Test Decision Tool

An interactive web-based tool that helps researchers and data analysts choose the correct statistical test in under 60 seconds.

## Purpose

Selecting the right statistical test is crucial. This tool simplifies the decision-making process by asking a few key questions about your data and research design.

## Features

- **Clean, minimal interface** - No distractions, just clarity
- **Interactive decision tree** - Guided questions lead to the right test
- **Comprehensive coverage** - Covers parametric and non-parametric tests
- **Real examples** - Each test includes practical use cases
- **Mobile responsive** - Works on any device
- 

## Usage

Simply open `index.html` in any modern web browser. No server or dependencies required.



##  Supported Tests

### Parametric Tests
- One-Sample T-Test
- Independent Samples T-Test
- Paired T-Test
- ANOVA
- Pearson Correlation & Regression

### Non-Parametric Tests
- Mann-Whitney U Test
- Wilcoxon Signed-Rank Test
- Kruskal-Wallis H Test
- Spearman Correlation

### Chi-Square Tests
- Goodness of Fit
- Test of Independence
- Test of Homogeneity

##  Based on Research

This tool is based on the methodology described in:

Pirani, S. (2024). Simplifying Statistical Decision Making: A Research Scholar's Guide to Parametric and Non-Parametric Methods. *International Journal of Multidisciplinary Research & Reviews*, Vol 03, No. 03, pp. 184-192.
